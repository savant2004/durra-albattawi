# Durrat Al-Battawi — درة البطاوي

Bilingual (English / Arabic) presentation website for Durrat Al-Battawi, an Iraqi-American car import business sourcing vehicles from US auctions (Copart, IAAI) for shipment to Iraq.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for production

```bash
npm run build
npm run preview
```

The production build is output to `dist/`.

## Tech stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui components
- Framer Motion
- Wouter (routing)
